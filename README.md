# TrustScan

TrustScan is a deepfake detection web app that helps users identify whether a video or image is authentic or artificially manipulated. Simply upload a media file, and TrustScan runs AI-powered analysis to return a confidence score and a detailed breakdown of the result.

## Features

- Upload videos or images (MP4, MOV, JPG, PNG) via drag-and-drop or file browser
- Real-time scanning with progress feedback
- Detection results with confidence scores and analysis breakdown
- Dashboard showing total scans, fake detections, and accuracy stats
- Scan history and investigation tracking
- User profile and settings

## Tech Stack

- **React 19** with TypeScript
- **Vite** for builds
- **React Router v7** for navigation
- **Zustand** for state management
- **TanStack Query** for async operations
- **Tailwind CSS v4** for styling
